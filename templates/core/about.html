Acerca de

Web realizada por Ezequiel Rodríguez como proyecto final para el curso de programación en python de Coderhouse. No utilice bootstrap, aproveché la opurtunidad y aprendí un poco mas sobre HTML y CSS, todo el proyecto fue hecho de cero. Me hubiese gustado dejarlo un poco mas bello pero no llegué con el tiempo.
Nota: En el video verán que hay imagenes pero por alguna razón no se suben a GitHub :(
Características del proyecto:
  • Web estilo red social (twitter, reddit) con creacion de usuarios, posteos y comentarios.
  • Posibilidad de crear, editar y eliminar posteos (solo si los posteos son del usuario o si el usuario es administrador).
  • Posibilidad de crear y eliminar comentarios (solo si los comentarios son del usuario o si el usuario es administrador).
  • Cada usuario tiene un perfil donde figuran todos sus posteos y comentarios recientes.
  • Los posteos estan catalogados por temas (topics).
  • Filtro por temas y sección de actividad reciente en la web.
